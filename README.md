# Bank Customer Analytics Dashboard
                                                            
# Project Overview
This Power BI dashboard analyzes customer demographics, loan adoption patterns, transaction behavior, and credit card performance to derive actionable business insights.
# Tools Used
- Power BI
- DAX
- Data Modeling 
- Data Visualization
# Dashboard 1: Customer Overview
- 10K Total Customers
- 1,310 Loan Customers
- Average Income Analysis
- Customer Segmentation by Education & Marital Status
# Key Insights:
- Married customers form the largest segment.
- Businessman category has highest average income.
- Loan adoption is relatively low compared to total customer base.
# Dashboard 2: Transaction & Credit Analysis
- 667K Total Transactions
- 46M Total Transaction Volume
- Q4 shows highest transaction growth.
- Blue Card category dominates transaction volume.
# Key Insights:
- Transaction spike observed in Q4.
- Chip usage is the most preferred transaction method.
- Interest earnings consistent across quarters.
